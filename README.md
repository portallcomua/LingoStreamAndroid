# 🛸 LingoStream AI

**AI Live Screen Subtitle Translator**

LingoStream AI — це додаток для перекладу субтитрів у реальному часі з екрану. Підтримує переклад англійських субтитрів з відео на YouTube та інших платформах.

---

## 📱 Features

- ✅ Автооновлення через GitHub Releases
- ✅ Збереження власних відео
- ✅ Категорії: Rock, Movies, Cartoons, TED, Навчання, Українські
- ✅ Переклад субтитрів (OCR екран)
- ✅ Українська / English
- ✅ Налаштування: мова, версія, Privacy Policy
- ✅ Підготовка до Google Play

---

### Android

2. Встановіть на пристрій

git clone https://github.com/portallcomua/LingoStreamAndroid.git
